App for managing PR lists

Angular Typescript Nodejs MongoDB TailwindCSS DaisyUI

DiscoApp è un’app full stack sviluppata in Angular, Node.js e MongoDB per semplificare la gestione delle liste nei locali. Permette ai PR di creare e gestire liste digitali, inviare QR code via SMS (Twilio) e validare l’ingresso con QR scanner all’entrata. Include autenticazione multi-ruolo, CRUD operations per eventi e utenti, e un’interfaccia curata con TailwindCSS + DaisyUI.
